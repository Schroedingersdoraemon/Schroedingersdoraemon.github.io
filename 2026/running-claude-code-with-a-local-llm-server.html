Running Claude Code with a Local LLM Server

2026-05-05 09:00

Today, May 5, 2026, we will explore how to run Claude Code using a local Large Language Model (LLM) server. This setup can be particularly useful for development, testing, or running Claude in an offline environment.

Prerequisites

Before you begin, ensure you have the following: - A machine with Python installed. - Access to a terminal or command prompt. - Basic knowledge of using Git and shell commands.

Step 1: Install Required Packages

First, let's install the necessary packages required to run Claude Code locally. We will use pip to install these packages:

pip install claudie-code[local]

This command installs Claude Code along with any dependencies needed for running it on a local server.

Step 2: Start the Local Server

Next, we need to start the local LLM server. Navigate to the directory where you have your project files and run the following command:

claudie-server start --host localhost --port 5000

This command starts the local server on localhost at port 5000. You can change the host and port as needed.

Step 3: Configure Claude Code to Use the Local Server

Now that the server is running, you need to configure Claude Code to use it. Open your terminal or command prompt and run:

claudie config set local-server-url http://localhost:5000

This command sets the local server URL for Claude Code.

Step 4: Test the Local Server

To ensure that everything is working correctly, you can test the local server by running a simple command. For example:

claudie run "What's the weather like in Paris?"

If everything is set up correctly, you should receive the current weather information for Paris.

Step 5: Stop the Local Server

When you are done working with the local server, you can stop it by running:

claudie-server stop

This command stops the local server gracefully.

Conclusion

Running Claude Code with a local LLM server is a powerful way to develop and test Claude in your own environment. With this setup, you have full control over your development workflow and can easily switch between local and remote environments as needed.

If you encounter any issues or need further assistance, feel free to refer to the Claude Code documentation for more detailed information.

Happy coding!
